# Query Sage

A minimalist terminal interface for natural language database orchestration. Speak to your data in plain English.

## Features

- Natural language SQL query generation
- AI-powered database interaction
- Terminal-based interface
- Support for multiple databases
- Real-time SQL feedback

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to start using Query Sage.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase
- Gemini AI
